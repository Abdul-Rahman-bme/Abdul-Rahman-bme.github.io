## 🚀 How to Update Your GitHub Repository

### Option 1: GitHub Web Interface (Easiest)

1. **Go to your repository**: `https://github.com/YOUR_USERNAME/YOUR_REPO`

2. **Update index.html**:
   - Click on `index.html` in the file list
   - Click the pencil icon (Edit this file)
   - Delete all content and paste the new `index.html` content
   - Scroll down and click "Commit changes"

3. **Create styles.css**:
   - Click "Add file" → "Create new file"
   - Name it `styles.css`
   - Paste the CSS content
   - Click "Commit new file"

4. **Create script.js**:
   - Click "Add file" → "Create new file"
   - Name it `script.js`
   - Paste the JavaScript content
   - Click "Commit new file"

5. **Keep existing files**:
   - `Abdul_Rahman_CV.pdf` - No changes needed
   - `profile.jpg` - No changes needed

### Option 2: Git Command Line

1. **Clone your repository** (if not already on your computer):
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
   cd YOUR_REPO
   ```

2. **Replace index.html**:
   - Delete the old `index.html`
   - Copy the new `index.html` from the files provided

3. **Add new files**:
   - Copy `styles.css` to your repository folder
   - Copy `script.js` to your repository folder

4. **Commit and push changes**:
   ```bash
   git add .
   git commit -m "Major portfolio update: modernized design, added new projects and achievements"
   git push origin main
   ```

   Note: Replace `main` with `master` if your default branch is named `master`

### Option 3: GitHub Desktop (User-Friendly)

1. Open GitHub Desktop
2. Select your repository
3. Replace `index.html` with the new version
4. Add `styles.css` and `script.js` files
5. Write commit message: "Major portfolio update"
6. Click "Commit to main"
7. Click "Push origin"

## 🐛 Troubleshooting

**Theme not switching?**
- Clear browser cache
- Check browser console for errors

**Animations not working?**
- Ensure JavaScript is enabled
- Check for console errors

**Layout broken on mobile?**
- Clear cache and refresh
- Try different browser

**GitHub Pages not updating?**
- Wait 2-3 minutes after push
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Check GitHub Actions tab for build status
